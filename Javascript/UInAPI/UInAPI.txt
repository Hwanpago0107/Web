1. 레퍼런스와 튜토리얼

프로그래밍을 공부하기 위한 자료는 크게 레퍼런스와 튜토리얼이 있다. 
통상 튜토리얼은 언어의 문법을 설명하고 레퍼런스는 명령어의 사전을 의미한다. 
ex) 튜토리얼 : 프로그래밍 관련 책, 수업 등
    레퍼런스 : 명령어들의 집합으로 구성된 서적, 사전, 리스트 등

2. 자바 스크립트의 API

자바스크립트 자체의 API 와 자바스크립트가 동작하는 호스트 환경의 API로 구분

자바스크립트 자체의 API : Math,정규표현식 등 기본적으로 제공됨
자바스크립트가 동작하는 호스트 환경 : 웹 브라우저, 서버(Node.js), googleApps 등

3. 자바스크립트 자체 API

- ECMAScript (표준문서)
- javascript 사전 (생활코딩 등)
- javascript 레퍼런스 (MDN)
- jscript 레퍼런스 (MSDN 마이크로소프트에서 제공)

4. 호스트 환경의 API

- 웹브라우저의 API
- Node.js 의 API
- googleApps scripts 의 API
